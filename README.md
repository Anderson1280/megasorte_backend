# MegaSorte Backend

[Português](#português) | [English](#english) | [Español](#español)

---

## Português
Backend robusto desenvolvido em FastAPI para o ecossistema MegaSorte.
* **Funcionalidades**: Autenticação JWT, controle de jogadas e integração com webhooks da Kiwify.
* **Tecnologias**: Python, FastAPI, SQLite e modelos de análise de dados.
* **Estrutura**: Organizado em módulos de autenticação (`auth`) e análise (`analytics`).

### Como rodar
1. Instale as dependências: `pip install -r requirements.txt`.
2. Configure o arquivo `.env` com suas chaves.
3. Inicie o servidor: `uvicorn app:app --reload`.
